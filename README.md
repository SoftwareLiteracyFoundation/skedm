skedm - scikit-learn compatible Empirical Dynamic Modeling
==========================================================

**skedm** is a [scikit-learn](https://scikit-learn.org) compatible implementation of [Empirical Dynamic Modeling (EDM)](https://en.wikipedia.org/wiki/Empirical_dynamic_modeling). Core code is based on the [pyEDM](https://pypi.org/project/pyEDM/) package. An in-depth introduction to EDM is in the [pyEDM Docs](https://sugiharalab.github.io/EDM_Documentation/).

See the [skedm documenation](https://sk-edm.readthedocs.io/en/latest/)
