###############
Getting started
###############


scikit-learn compatible EDM
===========================

Download sciedm
--------------

Clone the ``sciedm`` repository:

.. prompt:: bash $

  git clone https://github.com/SoftwareLiteracyFoundation/sciedm

Install `sciedm`:

.. prompt:: bash $

  cd sciedm
  python -m pip install .
